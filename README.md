# Кулинарная книга бабушки Марии 👵

Бабушка Мария Ивановна 50 лет вела тетрадку с рецептами.
Мы решили оцифровать всё это сокровище — чтобы вся семья
могла читать, и рецепты никогда не терялись.

## Рецепты

```
recipes/
├── soups/      — супы
├── desserts/   — выпечка и сладкое
└── salads/     — салаты и закуски
```

## Как вносить изменения

Работаем через Git: каждый рецепт — в своей ветке.
Готово → сливаем в main. Незавершённое держим в feature-ветках.
